    DSC A11Y Handoff v2.2.0
    Como usar
    COMEÇANDO
    ⬡ Seleção no canvas
    1

    Clique no componente no canvas (COMPONENT, COMPONENT_SET, INSTANCE ou FRAME).

    2

    Segure Shift e clique no template de handoff para selecioná-los juntos.

    3

    O plugin trava o contexto automaticamente e carrega os dados salvos anteriormente.

    ABAS
    G Geral

    Marque as plataformas do componente (Web, Mobile iOS, Android…) e os tipos de zoom aplicáveis.

    Use a busca para adicionar mapeamentos de teclado e gestos da biblioteca DSC.

    T Área de Toque

    Crie variações de configuração (estado padrão, expandido, etc.). Para cada variação, clique "Marcar área" e ajuste o frame rosa no canvas sobre o alvo de toque.

    Escolha o preset de tamanho (Aprimorado 44px, Mínimo 24px, etc.) ou insira dimensões livres.

    ↹ Tabulação

    Crie variações de configuração. Selecione os elementos focáveis diretamente no canvas e clique "Adicionar seleção" para definir a ordem de tabulação.

    ◉ Leitor de Tela

    Crie variações de configuração. Adicione conectores para cada elemento acessível, escolhendo o tipo de role ARIA e posicionando no canvas.

    Para agrupamentos, use "Marcar área de agrupamento" e ajuste o frame no canvas sobre a região a agrupar.

    ↗ Handoff

    Revise o resumo de todas as seções preenchidas e clique "Gerar Handoff" (primeira vez) ou "Atualizar Handoff" (atualizações).

    ⚙ Configurações

    Acesse o ⚙ no topo para ajustar o comportamento do plugin ao atualizar o handoff.

    Seções para atualizar

    Desmarque as seções que não devem ser regeradas. O conteúdo existente nelas será preservado com todas as edições manuais.

    Preservação de edições

    Mantém textos editados manualmente no handoff (descrições, observações, notas). Posicionamento e estrutura são sempre regerados.

    Sincronização de template

    Sincroniza a estrutura visual com o template original da biblioteca DSC ao gerar o handoff.

    MIGRAÇÃO
    ↻ Handoff antigo

    Se o handoff foi criado no formato anterior, o plugin detecta automaticamente e exibe o banner "Handoff Antigo Detectado".

    Clique "Importar Tudo" para migrar os dados. Os dados ficam salvos — na próxima abertura o formulário já vem preenchido.

    Ao clicar "Atualizar Handoff", o template antigo é substituído automaticamente pelo novo, preservando o conteúdo das seções não marcadas para atualizar.

    SOBRE
    A DSC A11Y Handoff

    Plugin Figma para preenchimento de templates de handoff de acessibilidade do Design System Caixa (DSC).

    Os dados de mapeamento de teclado, gestos e roles de leitor de tela são carregados automaticamente da camada plugin data do template.

    Configurações
    COMPORTAMENTO
    ↻ Sincronização de template

    Sincroniza a estrutura visual com o template original.

    🔗 Preservação de edições

    Ao atualizar, preserva textos editados manualmente no handoff (descrições, observações, notas). Alterações de posicionamento, conectores e estrutura são sempre regeradas com os dados atuais do plugin.

    ≡ Seções para atualizar

    Seções desmarcadas serão preservadas com todas as edições manuais.
