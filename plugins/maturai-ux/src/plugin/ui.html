Como usar

O que é o Maturai UX?

Plugin de coleta de dados de maturidade de UX. Estrutura o processo de discovery e análise dos projetos para alimentar a vertical de métricas de UX.

1

Preencha o Briefing

Registre o contexto, objetivos e escopo do projeto com campos padronizados.

2

Insira frameworks no canvas

Escolha um framework na lista e clique "Inserir no Canvas". Um template estruturado será criado no Figma com campos nomeados.

3

Preencha os campos no Figma

Edite os textos dentro do frame gerado. Cada campo tem um nome semântico que o scanner reconhece.

4

Escaneie o canvas

Clique em "Escanear" para extrair os dados preenchidos e estruturá-los no plugin.

5

Exporte para o HANDEX

Use o botão "Exportar Dados" para gerar um JSON que pode ser importado no HANDEX ou enviado para análise pela vertical.
