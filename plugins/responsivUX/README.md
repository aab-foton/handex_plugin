# ResponsivUX – Plugin Figma

Um plugin Figma para gerar layouts responsivos automaticamente em múltiplos breakpoints.

## 📋 Recursos

- 🎯 Geração de múltiplos breakpoints (de 768px até 3840px)
- 🔄 Auto Layout responsivo
- 📐 Constraints inteligentes
- 🎨 Interface integrada com design CAIXA
- 🌙 Modo claro/escuro
- 📱 Interface redimensionável com zoom

## 🚀 Como Usar

1. Selecione um Frame no Figma
2. Clique em um breakpoint específico ou "Gerar Todos" para criar versões responsivas
3. Configure o espaçamento entre frames com o campo "Gap horizontal"

## 📦 Instalação

- Adicione o arquivo manifest.json no Figma Plugin Manager
- Certifique-se de ter os arquivos: `code.js`, `ui.html`, `manifest.json`

## 🛠️ Desenvolvimento

```bash
npm install
npm run build      # Compila TypeScript para JavaScript
npm run watch      # Modo de desenvolvimento com watch
```
