UX Brain — Centro de Inteligência de UX

Este plugin atua como o cérebro investigativo dos seus arquivos no Figma. Ele lê o contexto de UI, extrai tokens e pode te elucidar fluxos através da IA.

Para que serve?

Qualquer pessoa que queira extrair dados de um projeto para gerar informações, insights ou insumos estratégicos pode usar este plugin. Não é sobre exportar telas — é sobre criar um pacote de dado com metadados, contexto e referências que qualquer sistema consiga interpretar.

Por que não fazer isso pelo Figma nativo?

O Figma permite exportar telas e copiar links. Mas nenhuma exportação nativa lê tokens, bibliotecas, anotações, textos internos e componentes ao mesmo tempo — e menos ainda organiza isso num arquivo semântico pronto para ser consumido por uma IA.

🧠 Criando um RAG

O pacote gerado aqui foi desenhado para ser o alimento de Agentes de IA. Quando esses arquivos são adicionados a um motor de IA (ChatGPT, Gemini, Claude ou um agente interno da CAIXA), ele passa a responder com precisão cirúrgica sobre esse projeto — porque terá contexto real, não suposições. Isso é um RAG (Retrieval-Augmented Generation) aplicado ao UX.

Instruções de Uso

1

Copie o Contexto

Clique no botão "Copiar Contexto" para levar todos os metadados do projeto para sua área de transferência.

2

Cole na sua IA de preferência

Abra o Copilot, Gemini ou ChatGPT e cole o texto. Isso dará à IA o conhecimento "especialista" sobre seu arquivo Figma.

3

Pergunte Estratégicamente

Tente perguntas como: "Baseado no contexto, quais regras de negócio estão implícitas?" ou "Gere o código CSS seguindo os tokens de cor fornecidos."
